# **App Name**: KULOOC

## Core Features:

- Bilingual Interface: Automatically detect user's location in Quebec and prioritize French, with real-time translation of addresses. The tool can reason whether translation is necessary based on user location.
- Quick Scan Address: Use the device's camera to scan street numbers for precise address input.
- Driver Match by Affinity: Intelligently match users with drivers based on language, eco-friendliness (electric vehicle preference), rating, and local proximity.
- Price Lock: Allow users to lock in a price to avoid surge pricing for a short period.
- Interactive Map: Display real-time locations of drivers, points of interest (restaurants, charging stations), and high-demand zones using Mapbox with 3D buildings.
- Dynamic Configuration: Fetch app configurations like active services, promotions, and language settings from Firestore to enable real-time updates.
- User Preferences: Store user preferences, favorite services, and frequent addresses in Firestore for personalized experiences.

## Style Guidelines:

- Primary color: Deep blue (#1A237E), evoking trust, safety, and a connection to Canadian heritage (think RCMP). The color symbolizes reliability and stability.
- Background color: Very light gray (#F5F5F5), to give a modern, clean, and spacious feel.
- Accent color: Electric blue (#2962FF) for interactive elements like buttons and links, ensuring high visibility.
- Body and headline font: 'Inter', a sans-serif font, offering a clean and modern aesthetic suitable for a tech-forward application.
- Use clear, minimalist icons to represent services and actions. Icons should be easily understandable in both French and English contexts.
- Mobile-first design with a focus on intuitive navigation. Prioritize key actions and information on the home screen, like the search bar and service options.
- Use subtle animations for transitions and feedback, such as map marker movements or loading states. Animations should be smooth and responsive to enhance user experience.
